A New Hope

Bienvenue chez BeCode. La nouvelle formation en horaire de jour pour apprendre les technologies numériques. Gratuitement.

Le 20 septembre 2018, Nicolas découvre l'exitence de BeCode.

Il est convaincu que cette formation est faite pour lui.

Le jour-meme, il pose sa candidature sur le site web de BeCode.

Quelques temps après, il reçoit la confirmation de sa sélection pour la phase finale.

Après avoir passé différents tests, le voici désormais sélectionné pour l'aventure BeCode.

Son unique but dans cette formation est de devenir développeur web pour pouvoir réaliser son reve.

Nicolas veut travailler en tant que freelance et ainsi pouvoir etre libre pour voyager dans tout l'univers.
